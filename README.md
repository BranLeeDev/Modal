# Modal

Un modal sencillo creado con HTML, CSS y JavaScript. Se sigue el enfoque de diseño Desktop First y se emplea la metodología BEM.

## Características

- Responsive
- BEM (Block Element Modify)
- Desktop First
- JavaScript Vanilla
